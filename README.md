# 🚀 X'perienced – Experience Sharing Platform

## 📌 Overview

**X'perienced** ek modern MERN stack based platform hai jaha users apne **study experiences share kar sakte hain**, dusre students ke posts dekh sakte hain, aur interact kar sakte hain.

Yeh platform specially students ke liye banaya gaya hai jaha wo apne learnings, mistakes aur strategies share kar sake.

---

## ✨ Features

### 🔐 Authentication

* User Signup & Login
* Secure JWT Authentication
* Protected Routes

### 📝 Post System

* Create Experience Posts
* View All Posts
* Like ❤️ Posts
* Save 🔖 Posts

### 💬 Interaction

* Comment on Posts
* Reply System
* User Engagement Features

### 🎨 UI/UX

* Clean Black & White Theme
* Responsive Design
* Modern Card Layout
* Smooth Animations

### 👤 User Profile

* View User Details
* Saved Posts Section
* Activity Tracking

## 🚀 Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/xperienced.git
cd xperienced
```

### 2️⃣ Backend Setup

```bash
cd backend
npm install
npm start
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## 🔗 API Endpoints

### 👤 Auth Routes

* POST `/api/auth/register`
* POST `/api/auth/login`

### 📝 Post Routes

* GET `/api/posts`
* POST `/api/posts`
* PUT `/api/posts/like/:id`
* PUT `/api/posts/save/:id`
